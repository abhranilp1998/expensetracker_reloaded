2. Scroll to Animation Settings
3. Select "Slide Left" animation
4. Disable animations if needed
5. Go back to see new animation style

2. Scroll to Feature Settings
3. Toggle Notifications ON/OFF
4. Toggle Haptic Feedback ON/OFF
5. All changes saved

1. Open Settings
2. Scroll to bottom
3. Tap "Reset to Defaults"

5. App resets to factory settings
6. All transactions preserved
